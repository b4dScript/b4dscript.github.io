Hack The Box | Reactor

Linux - Easy

Reactor is an easy Linux machine featuring Next.js 15.0.3 vulnerable to CVE-2025-55182 (React2Shell), leading to unauthenticated RCE. Lateral movement via SQLite credential dumping and MD5 cracking leads to a Node.js Inspector debug port for root privilege escalation.

Hack The Box | DevHub

Linux - Medium

DevHub is a medium Linux machine exploiting CVE-2026-23744 in MCPJam Inspector v1.4.2 for unauthenticated RCE. Internal enumeration reveals a Jupyter notebook and an OPSMCP API, leading to credential dumping and root via SSH key.

Hack The Box | Silo

Windows - Medium

Silo focuses mainly on leveraging Oracle to obtain a shell and escalate privileges. It was intended to be completed manually using various tools, however Oracle Database Attack Tool greatly simplifies the process, reducing the difficulty of the machine substantially.
